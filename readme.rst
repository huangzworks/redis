关于本文档
==================

在线阅读
---------------------

本文档可以在 `redis.readthedocs.org <http://redis.readthedocs.org>`_ 在线阅读。

翻译事项 
----------------------

在 Redis 官方文档的基础上，本文档做了以下改进：

- 修改了官方文档一些拗口的地方，统一了文风(官方文档是多人协作)，对于一些不好的例子进行了结构重排甚至重写(比如 sort 命令)
- 所有命令都带有代码示例，补齐了一些官方文档没有触及到的地方
- 和 Redis 官方的文档更新保持一致，同步翻译成中文

希望这个译文能对各位喜欢 Redis 的朋友们有所帮助。
